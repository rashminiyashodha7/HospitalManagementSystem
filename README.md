# HospitalManagementSystem
This project is a hospital management system project using c#.net
The project has admin login part. After login, the admin can access to the main system. The login is developed using c# and sql database. Wrong username and password will display a message.
Features of the system
1) Admin Login 
2) Department Management
3) Patient Management
4) Doctor Management
5) Nurse Management
6)Appointment Management
7) Payment Management
